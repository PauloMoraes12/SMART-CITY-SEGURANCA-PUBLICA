# Smart City - Segurança Pública Inteligente

## Descrição

Projeto acadêmico desenvolvido com foco em modelagem de dados para uma plataforma de segurança pública inteligente da Cidade Alfa.

O objetivo do projeto é estruturar uma solução capaz de registrar ocorrências, monitorar regiões de risco, integrar pontos policiais, iluminação pública, câmeras inteligentes e análise estatística regional.

O projeto foi construído utilizando conceitos de:

- Modelagem Conceitual
- Modelagem Lógica
- Modelagem Relacional/Física
- Regras de Negócio
- Banco de Dados Relacional
- SQL

---

## Objetivos do Projeto

- Registrar ocorrências de emergência
- Direcionar atendimentos automaticamente
- Monitorar regiões com maior incidência de ocorrências
- Integrar câmeras inteligentes e sensores urbanos
- Gerar estatísticas e mapas de calor
- Auxiliar decisões estratégicas de segurança pública

---

## Tecnologias e Ferramentas

- SQL
- Oracle SQL Developer Data Modeler
- Oracle Database 11g
- Modelagem de Dados
- DDL SQL
